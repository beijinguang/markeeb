# mac按键符号



符号	说明
⌘	Command
⇧	Shift
⌥	Option
⌃	Control
↩︎	Return/Enter
⌫	Delete
⌦	向前删除键（Fn+Delete）
↑	上箭头
↓	下箭头
→	右箭头
←	左箭头
⇞	Page Up（Fn+↑）
⇟	Page Down（Fn+↓）
Home	Fn + ←
End	Fn + →
⇥	右制表符（Tab键）
⇤	左制表符（Shift+Tab）
⎋	Escape (Esc)

## mac输入特殊符号的方法

⚠️ ⌘+⌃+space 可以直接调出输入表情和特殊符号的选框, 开始显示的是emoji表情, 一直往下面拉滑或者点右上角的图标就是各种特殊符号咯.
